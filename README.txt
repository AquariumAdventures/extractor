No need to install Python or any packages.

To run:

Double-click extractor.exe

You must set an environment variable: OPENAI_API_KEY with your valid OpenAI key.

The app should prompt for your API key on first run 



To set it permanently/manually on Windows:

Press Windows + R, type sysdm.cpl, and hit Enter.

Go to Advanced ‚Üí Environment Variables.

Under User variables, click New:

Name: OPENAI_API_KEY

Value: sk-... (your actual key)

Then just double-click extractor.exe to run the app.
